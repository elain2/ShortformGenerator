# Shorts 자동 생성 워크플로우

## 개요

감성 Vlog 스타일의 숏츠를 반자동으로 제작하는 파이프라인.
여러 영상에서 하이라이트를 추출하고, HyperFrames로 초안을 생성한 뒤, VLLO에서 더빙과 최종 편집을 진행한다.

---

## 워크플로우 다이어그램

```
┌─────────────────────────────────────────────────────────────────────┐
│                           INPUT                                      │
├─────────────────────────────────────────────────────────────────────┤
│  1. 원본 영상들 (여러 개)                                             │
│  2. 스크립트 (자막용 텍스트 + 타이밍 힌트)                             │
└─────────────────────────────────────────────────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    STEP 1: 하이라이트 추출                            │
├─────────────────────────────────────────────────────────────────────┤
│  도구: OpenCV, PySceneDetect, 또는 FFmpeg                            │
│                                                                      │
│  - 장면 전환 감지 (scene detection)                                   │
│  - 움직임/변화량 분석 (motion analysis)                               │
│  - 밝기/색상 변화 피크 감지                                           │
│  - 각 영상에서 N초 분량의 하이라이트 클립 추출                         │
│                                                                      │
│  출력: highlight_001.mp4, highlight_002.mp4, ...                     │
└─────────────────────────────────────────────────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    STEP 2: 음원 선택 및 준비                          │
├─────────────────────────────────────────────────────────────────────┤
│  소스: 로열티 프리 음원 사이트                                        │
│    - Pixabay Music                                                   │
│    - Free Music Archive                                              │
│    - Uppbeat (무료 티어)                                              │
│    - YouTube Audio Library                                           │
│                                                                      │
│  선택 기준:                                                          │
│    ✓ 잔잔한 분위기 (더빙 음성과 충돌 방지)                            │
│    ✓ Lo-fi, Ambient, Acoustic 장르 권장                              │
│    ✓ 보컬 없는 인스트루멘탈                                          │
│    ✓ BPM 60-90 정도의 느린 템포                                      │
│                                                                      │
│  출력: bgm.mp3 (볼륨 -12dB ~ -18dB 권장)                             │
└─────────────────────────────────────────────────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    STEP 3: 자막 타이밍 계산                           │
├─────────────────────────────────────────────────────────────────────┤
│  입력: 스크립트 텍스트                                                │
│                                                                      │
│  처리 로직:                                                          │
│    1. 스크립트를 문장/구 단위로 분리                                  │
│    2. 각 문장의 예상 더빙 시간 계산                                   │
│       - 한국어 기준: 약 3-4음절/초                                   │
│       - 또는 글자 수 × 0.15초 (여유 포함)                            │
│    3. 자막 노출 시간 = 더빙 예상 시간 + 0.5초 버퍼                    │
│    4. 전체 숏츠 길이에 맞게 타이밍 배분                               │
│                                                                      │
│  출력: subtitles.json                                                │
│  {                                                                   │
│    "segments": [                                                     │
│      { "text": "오늘 하루도", "start": 0, "duration": 2.0 },         │
│      { "text": "감사한 마음으로", "start": 2.0, "duration": 2.5 },   │
│      ...                                                             │
│    ]                                                                 │
│  }                                                                   │
└─────────────────────────────────────────────────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    STEP 4: HyperFrames 컴포지션                       │
├─────────────────────────────────────────────────────────────────────┤
│  HyperFrames HTML 구성:                                              │
│                                                                      │
│  composition/                                                        │
│  ├── index.html          # 메인 컴포지션                             │
│  ├── styles.css          # 자막 스타일, 전환 효과                     │
│  ├── assets/                                                         │
│  │   ├── highlights/     # 추출된 하이라이트 클립들                   │
│  │   └── bgm.mp3         # 배경 음악                                 │
│  └── subtitles.json      # 자막 데이터                               │
│                                                                      │
│  주요 구현:                                                          │
│    - 9:16 세로 비율 (1080x1920)                                      │
│    - 하이라이트 클립 시퀀스 배치                                      │
│    - 클립 간 전환 효과 (fade, crossfade)                             │
│    - 자막 애니메이션 (fade-in, 타이핑 효과 등)                        │
│    - BGM 볼륨 자동 조절 (더빙 구간 대비)                              │
│                                                                      │
│  렌더링: npx hyperframes render --output draft.mp4                   │
└─────────────────────────────────────────────────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    STEP 5: VLLO 최종 편집                             │
├─────────────────────────────────────────────────────────────────────┤
│  draft.mp4를 VLLO로 import 후:                                       │
│                                                                      │
│  수정 작업:                                                          │
│    □ 클립 순서/길이 미세 조정                                        │
│    □ 자막 위치/스타일 수정                                           │
│    □ 추가 효과 (스티커, 필터 등)                                     │
│    □ BGM 볼륨 조절                                                   │
│                                                                      │
│  더빙 작업:                                                          │
│    □ 자막에 맞춰 음성 녹음                                           │
│    □ 음성 볼륨 vs BGM 볼륨 밸런스 조절                                │
│    □ 필요시 음성 이펙트 적용                                         │
│                                                                      │
│  최종 Export: final_shorts.mp4                                       │
└─────────────────────────────────────────────────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────────────┐
│                           OUTPUT                                     │
├─────────────────────────────────────────────────────────────────────┤
│  완성된 숏츠 영상 (60초 이하, 9:16 비율)                              │
│  → YouTube Shorts / Instagram Reels / TikTok 업로드                  │
└─────────────────────────────────────────────────────────────────────┘
```

## 기술 스택 (예상)

| 단계 | 도구/기술 | 용도 |
|------|----------|------|
| 하이라이트 추출 | Python + OpenCV / PySceneDetect | 장면 분석, 클립 추출 |
| 음원 | Pixabay, Uppbeat 등 | 로열티 프리 BGM |
| 자막 처리 | Python 스크립트 | 타이밍 계산, JSON 생성 |
| 영상 합성 | HyperFrames (Node.js) | HTML → MP4 렌더링 |
| 최종 편집 | VLLO (iOS/Android) | 더빙, 미세 조정 |

## 설계 결정사항

### 하이라이트 추출
| 항목 | 결정 |
|------|------|
| 추출 기준 | 움직임 + 색감 + 피사체 감지 **세 가지 모두 사용**, 점수화하여 순위 결정 |
| 클립 길이 | 고정값 아님, **하이라이트 구간에 따라 가변** |
| 클립 개수 | **전체 숏츠 길이 대비 자동 계산** |

### 자막 타이밍
| 항목 | 결정 |
|------|------|
| 더빙 속도 | **파라미터로 설정 가능** (기본값: 3.5음절/초) |
| 영상-자막 싱크 | **draft.mp4 생성 시 자동 맞춤** |
| 강조 표시 | **안 함** (MVP 단계에서 제외) |

### HyperFrames 템플릿
| 항목 | 결정 |
|------|------|
| 템플릿 구조 | **안 함** (MVP 단계에서 제외) |
| 자막 스타일 | **감성 vlog 스타일 1개**로 시작 |
| 전환 효과 | **단순 컷전환** (트랜지션 없음) |

### VLLO 연동
| 항목 | 결정 |
|------|------|
| 더빙 가이드 | **안 넣음** |
| 자막 SRT 파일 | **별도 export 안 함**, draft.mp4에 감성 vlog 스타일로 렌더링하여 포함 |

## 예상 사용 시나리오

```bash
# 1. 하이라이트 추출
python extract_highlights.py ./input/videos/ --output ./output/highlights/ --clip-length 3

# 2. 자막 타이밍 생성
python generate_subtitles.py ./input/script.txt --speaking-rate 3.5 --output ./hyperframes/subtitles.json

# 3. HyperFrames 렌더링
cd hyperframes && npx hyperframes render --output ../output/draft.mp4

# 4. VLLO로 draft.mp4 열어서 최종 편집 + 더빙
```
